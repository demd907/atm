# 💰 간단한 입출금 프로그램



Python의 기본 입출력과 조건문, 반복문을 활용한 **간단한 은행 입출금 프로그램**입니다.  
사용자는 금액을 입금하거나 출금할 수 있으며, 잔액을 확인할 수 있습니다.



## 🧩 기능 안내

1. **입금**  
   - 금액을 입력하면 잔액에 더해집니다.
2. **출금**  
   - 출금 금액이 잔액보다 많으면 "잔액 부족" 메시지를 출력합니다.
   - 출금 가능한 경우, 해당 금액만큼 잔액에서 차감됩니다.
3. **영수증 보기** (아직 미구현 or 확장 예정)
4. **종료**  
   - 프로그램을 종료합니다.
